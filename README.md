# 小売物価統計の価格変動パターン分析

総務省の小売物価統計Excelファイル `b00113.xlsx` を使い、複数品目の月別価格推移を都市別に分析する。

目的は、単純な平均価格の集計ではなく、都市と品目の組み合わせごとに価格推移の特徴を抽出し、似た価格変動パターンをクラスタリングで分類することである。

## 分析の流れ

1. Excelの全シートを読み込む
2. 各シートの都市別・月別価格をロング形式に変換する
3. 都市×品目ごとに、月を説明変数、価格を目的変数とする線形回帰を行う
4. 回帰結果から価格変動の特徴量を作成する
5. 特徴量を標準化し、K-meansクラスタリングを行う
6. PCAで2次元に圧縮し、クラスタリング結果を可視化する
7. クラスターごとの特徴を解釈する

## 実行方法

必要なライブラリをインストールする。

```bash
python -m pip install -r requirements.txt
```

特徴量を作成する。

```bash
python src/price_trend_regression.py
```

クラスタリングを行う。

```bash
python src/price_pattern_clustering.py
```

## 入力データ

入力ファイルはプロジェクト直下の `b00113.xlsx`。

各シートは1つの品目として扱う。ただし、すしは元データの区分を維持するため、以下のように分けている。

- `2121` で始まるシート: `2121_すし`
- `2123` で始まるシート: `2123_すし`
- それ以外: シート名をそのまま分析項目名にする

## 作成する特徴量

`src/price_trend_regression.py` では、都市×品目ごとに線形回帰を行い、以下の特徴量を作る。

- `データ数`
- `1月価格_円`
- `12月価格_円`
- `1月から12月の変化額_円`
- `1か月あたりの変化額_円`
- `年間換算の変化額_円`
- `決定係数`

クラスタリングでは、このうち以下の3つを使う。

- `1月価格_円`: 価格水準の高さ
- `1か月あたりの変化額_円`: 価格が上がっているか、下がっているか、どれくらい変化しているか
- `決定係数`: 価格変化がどれくらい直線的か

## 出力ファイル

特徴量作成の出力:

- `output/price_trend_features_output/全シート_項目別_都市別_線形回帰トレンド分析.csv`

クラスタリングの出力:

- `output/price_pattern_clustering_output/都市別品目別_クラスタリング結果_PCA付き.csv`

## 考察

### クラスター0: 安定した上昇グループ

クラスター0は、比較的低〜中価格帯で、価格が安定して上昇しているグループである。平均1月価格は約750円、平均12月価格は約791円で、1月から12月にかけて平均で約41円上昇している。平均決定係数も0.72と高く、月ごとの価格推移が比較的直線的である。

品目構成を見ると、`2101うどん（外食）`、`2133カレーライス（外食）`、`2102中華そば（外食）` など、外食系の品目が多い。外食品目は、対象期間中に比較的一貫した値上がり傾向を示していると考えられる。

### クラスター1: 月ごとの変動が大きい緩やかな上昇グループ

クラスター1は、平均では価格が上昇しているものの、決定係数が低いグループである。平均1月価格は約788円、平均12月価格は約812円で、平均変化額は約25円の上昇にとどまる。平均決定係数は0.24と低く、価格変化が直線的というより、月ごとの上下動が大きい傾向がある。

品目構成では、`2027ビール風アルコール飲料`、`2021ビール`、`2041ワイン`、`2026発泡酒` など、酒類が多く含まれている。酒類は、外食系と比べると価格推移が一定方向に安定して動くというより、月ごとの価格変動が大きい可能性がある。

### クラスター2: 高価格帯で価格低下傾向のグループ

クラスター2は、高価格帯で、平均的には価格が低下しているグループである。平均1月価格は約1684円、平均12月価格は約1644円で、1月から12月にかけて平均で約41円低下している。

このクラスターの中心は `2033ウイスキー` で、81件が含まれている。ほかに `2121_すし` も一部含まれる。全品目が一様に値上がりしているわけではなく、高価格帯の一部品目では価格低下傾向も見られる。

ただし、クラスター2には個別には上昇している都市×品目も含まれる。これはK-meansが、価格水準、月次変化額、決定係数の3特徴量をまとめて見て分類しているためである。したがって、クラスター名は「平均的な特徴」として読む必要がある。

### クラスター3: 例外的な急上昇グループ

クラスター3は、5件だけの小さなグループで、すべて `2121_すし` である。平均1月価格は約1534円、平均12月価格は約1998円で、平均変化額は約464円と非常に大きい。平均決定係数も0.81と高く、急激な上昇が比較的一貫して起きている。

含まれる都市は、奈良市、長崎市、広島市、富山市、和歌山市である。件数が少ないため、全体傾向というより、外れ値または例外的な価格変動パターンとして扱うのが安全である。

## PCA散布図の見方

`クラスタリング結果_PCA散布図.png` では、1点が1つの「都市×品目」を表す。

たとえば、以下のような単位で点が配置されている。

- `2021ビール × 札幌市`
- `2101うどん（外食） × 函館市`
- `2121_すし × 奈良市`
- `2033ウイスキー × 青森市`

横軸の主成分1、縦軸の主成分2は、元の列そのものではない。クラスタリングに使った `1月価格_円`、`1か月あたりの変化額_円`、`決定係数` の3特徴量を、PCAによって2次元に圧縮した軸である。

点が近いほど、価格水準や価格変動傾向が似ている。色が同じ点は同じクラスターに分類されている。

## 全体のまとめ

今回の分析から、対象品目の価格変動は単純な一様上昇ではなく、複数のパターンに分かれることが分かった。

- 外食系の品目は、比較的安定した上昇傾向を示しやすい
- 酒類は、月ごとの価格変動が大きい傾向がある
- ウイスキーを中心とする高価格帯の一部では、価格低下傾向が見られる
- すしの一部都市では、例外的な急上昇が見られる
